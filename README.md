A To-Do List Web Application is a productivity-based web project that allows users to organize their daily tasks efficiently.
This project is developed using HTML, CSS, and JavaScript and demonstrates the core concepts of frontend web development such as structure, styling, and dynamic interaction.

The application allows users to:
-Add tasks
-Mark tasks as completed
-Highlight important tasks
-Delete tasks
-Track progress visually
The project is implemented as a single-page application, meaning all the code is written in one HTML file for simplicity and ease of understanding.
